Showcard

The primary content tile across all Spectrum Video surfaces. Defined here as a single base component with configurable style, size, width, and DFTA type — replacing use-case-specific implementations with one semantically-sound, token-driven definition consumable by both designers and UI Node agents.

⬤  In Progress Molecule tvOS · Roku · JS/TVSDK · iOS · Android Deliverable 04
01

Overview

The showcard is the primary content tile used across all Spectrum Video surfaces. It appears in swimlanes, grids, guide rows, and search results. The card is composed of an image layer, an optional DFTA lockup (logo + eyebrow + title + metadata), optional left and right badges, an optional bookmark, and an optional progress bar.

Cards are named using a structured convention: card-[size]-[style]-[width]-[dfta-type]. For example: card-xl-flyout-wide-allDfta-logo or card-m-fill-wide-titleMetadata.

Style vs. variant: The card style (Fill, Flyout, Overlay) determines where the DFTA lockup sits relative to the art. The card size (XL, L, M, S) and width determine layout density. These are orthogonal axes — any size can use any style.

06

Usage — Do & Don't

✓ DOKeep art clean. Place all text as UI-layer elements using Foundation tokens, not embedded in the image.
✗ DON'TDuplicate text in both the art and the UI layer. When art has embedded title treatment, suppress the UI-layer title token.
✓ DOUse consistent 16:9 aspect ratio across a swimlane. All cards in the same row share the same variant.
✗ DON'TMix aspect ratios within the same swimlane. Use the movie (2:3) variant only in dedicated movie rows.
✓ DOShow the focus ring only when focus is active. Use the state-card-focused token — never custom override per component.
✗ DON'TCreate custom focus styles per card variant. All focus states must resolve from the Focus Ring atom and shared token.

Content Guidelines

Rules for copy within the showcard. These apply across all variants and platforms unless noted.

FieldMax lengthCasingTruncationNotes
Title 2 lines Title case Ellipsis at line 2 Use show/movie title as provided by Gracenote. Never abbreviate.
Eyebrow 1 line · ~30 chars All caps Truncate at container edge Default format: NETWORK · GENRE or time info. Restriction overrides: OOH → "CONNECT TO WATCH", Unentitled → "UPGRADE TO WATCH", TVOD → "AVAILABLE TO RENT". Recordings → "RECORDED DATE" / "RECORDING NOW". Amber (#C8A84B) color always.
State badge label 8 chars All caps Do not truncate — shorten copy instead LIVE, NEW, REC, RERUN, UPCOMING. Never use lowercase.
Metadata badge label 6 chars All caps Do not truncate 4K, HDR, CC. Price format: $4.99. Episode count: "3 Episodes".
Expiry string 1 line · ~20 chars Sentence case Truncate at container edge Format: "Expires in X days". Threshold for display: ≤7 days remaining.
